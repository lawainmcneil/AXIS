# SHERPA Axis

A single-page portfolio analyzer and retirement income planning prototype for financial advisors.

## Run

Open `sherpa-axis.html` in a browser for the SHERPA Axis portfolio analyzer and retirement income planning prototype.

Open `sherpa-intake.html` in a browser for the standalone SHERPA Axis client intake form.

The app uses Tailwind CSS and Chart.js from CDNs, with DFA model return data embedded locally in `sherpa-data.js`. The standalone intake passes client data into Axis through URL parameters.
